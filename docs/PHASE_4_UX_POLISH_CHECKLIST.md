# Fase 4 — Checklist de Polimento UX/UI (Alfabetização Ampla)

Este checklist orienta o ajuste final da experiência para estudantes em diferentes estágios de alfabetização, sem restringir por faixa etária fixa.

## 1) Legibilidade e compreensão

- [ ] Fontes e pesos mantêm leitura clara em telas pequenas e grandes.
- [ ] Termos da interface são simples, diretos e consistentes entre telas.
- [ ] Rótulos de ação usam verbos objetivos (`Ouvir`, `Salvar`, `Jogar`, `Voltar`).
- [ ] Mensagens de feedback ajudam a tentar novamente sem tom punitivo.

## 2) Toque e interação (tablet e desktop)

- [ ] Botões principais têm área de toque confortável (mínimo visual equivalente a ~44px).
- [ ] Cartas arrastáveis respondem com feedback imediato de “pegada”.
- [ ] Soltar fora da zona retorna com animação suave e previsível.
- [ ] Não há cliques acidentais críticos durante arraste/navegação.

## 3) Contraste e hierarquia visual

- [ ] CTAs principais têm contraste forte com o fundo.
- [ ] Estados de acerto, erro e bloqueio são distintos e intuitivos.
- [ ] Vogais e consoantes possuem diferenciação visual estável em todas as telas.
- [ ] Informações pedagógicas e de progresso têm prioridade visual adequada.

## 4) Feedback multimodal (som + movimento + visual)

- [ ] `GOL!` gera celebração consistente (texto, confete e áudio).
- [ ] `Quase lá` orienta nova tentativa com incentivo.
- [ ] Sons de UI (`tap`, `swish`) ocorrem sem sobrecarga auditiva.
- [ ] `prefers-reduced-motion` mantém usabilidade sem remover clareza de estado.

## 5) Fluxo pedagógico contínuo

- [ ] Loop `Vestiário -> Campo/Match -> Álbum` mantém progressão compreensível.
- [ ] `Prancheta` e `Campeonato` reforçam autoria e prática fonêmica.
- [ ] `Boletim` mantém leitura clara para acompanhamento docente.
- [ ] Persistência local não perde progresso ao alternar telas.

## 6) Critérios de aceite da fase

Marque **GO** apenas se:

- [ ] `npm run lint` concluído sem erros.
- [ ] `npm run build` concluído com sucesso.
- [ ] Checklist acima validado em pelo menos 1 desktop e 1 tablet.
- [ ] Equipe pedagógica confirma aderência ao objetivo de alfabetização ampla.
